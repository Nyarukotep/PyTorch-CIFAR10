# CNN-on-CIFAR10
## Pytorch Implementation of Deep Learning Models/Algorithms

This repository is used to implement popular model architectures on the `CIFAR10` dataset.

## Environment

- CUDA Version: 10.2 

```
torch==1.5.0
torchvision==0.6.0
numpy==1.19.2
```
